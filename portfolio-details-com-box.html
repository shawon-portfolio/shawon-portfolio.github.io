Com-Box Details

Com-Box (Grievance Management tools)

Com-Box is a SaaS product designed to streamline grievance and feedback management for RMG brands and their suppliers globally. Trusted by renowned brands such as Marccain, Madness, Vaude, and Jack Wolfskin, it enables anonymous reporting of complaints, feedback, and suggestions while maintaining confidentiality between complainants and companies.

This advanced system supports 30 languages, including voice instructions, text, and file attachments, ensuring accessibility for a diverse user base. Users can scan QR codes displayed on supplier posters to access the complaint form directly, with key data fields pre-filled, simplifying the process.

Key Features:

  • Anonymous complaint submission for user privacy.
  • Issue categorization and human translation for accuracy.
  • Direct chat functionality between users and representatives.
  • Admin-only group chats for critical issue discussion, with file and audio attachments.
  • Complex but flexible user management system for role-based access.
  • QR code functionality for streamlined complaint submission.
  • Simplified admin dashboard for real-time monitoring and analysis.
  • Feedback option open to all users for broader inclusivity.

Tech Stack: Built with Angular for a dynamic front-end experience, .NET for robust back-end logic, and deployed on Azure for scalable cloud infrastructure. This combination ensures high availability, security, and performance.

Com-Box empowers brands to handle complex grievance management effectively, fostering trust and transparency between companies and their supply chains.
