Com-Box (Grievance Management tools)

Developed a multilingual communication tool, Com-Box, enabling users to anonymously report complaints, give feedback, or make suggestions to participating companies. The solution supports 30 languages with voice instructions, text, and file attachments. I implemented a chat function for direct user interaction and an advanced admin system with flexible access control. My work enhanced issue categorization, human translation, and enabled admin-only group chats for critical discussions. The tool simplifies feedback management, empowering companies to respond effectively to user concerns.

This system is built using Angular for the front-end, .NET for the back-end, and hosted on Azure for cloud infrastructure. Angular provides a dynamic and responsive user interface, while .NET handles the business logic and data processing on the server side. Azure offers scalable cloud services for deployment, ensuring high availability, security, and performance.
