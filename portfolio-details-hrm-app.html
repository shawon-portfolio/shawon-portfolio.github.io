HRM App Details

HRM App (Human Resource Management Tool)

The HRM software is a comprehensive solution designed to manage and streamline human resource processes, including employee management, payroll, attendance tracking, leave management, and performance reviews. It aims to enhance the efficiency of HR operations and improve employee experience by providing a user-friendly platform for all HR-related activities.

The system is built using Angular 16 (Standalone) for the front-end, utilizing the Vex Angular template for a clean and responsive UI. The back-end is powered by Spring Boot for scalable and robust performance, and hosted on AWS to ensure reliable cloud infrastructure and scalability.
